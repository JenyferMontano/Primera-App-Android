# LAB2 - Mi Primera App Android

Este es un proyecto sencillo de Android desarrollado como parte del Laboratorio 2, para el curso de Dispositivos Moviles, I Ciclo 2026.

## Características
- Interfaz construida enteramente con **Jetpack Compose**.
- Diseño basado en **Material Design 3**.
- Soporte para **Edge-to-Edge**.
- Estructura limpia y moderna en **Kotlin**.

## Tecnologías Utilizadas
- **Lenguaje:**KOTLIN
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Diseño:** Material 3
- **Build System:** Gradle (Kotlin DSL)

## Vista Previa
La aplicación muestra:
1. Un saludo principal: "¡Hola, Mundo! 🌍"
2. Un subtítulo: "Mi primera app Android"

## Requisitos
- Android Studio.
- JDK 17 o superior.
- Android SDK 34 o superior.
