# LAB2 - Mi Primera App Android

Este es un proyecto sencillo de Android desarrollado como parte del Laboratorio 2. La aplicación muestra un mensaje de bienvenida utilizando **Jetpack Compose**.

## 🚀 Características
- Interfaz construida enteramente con **Jetpack Compose**.
- Diseño basado en **Material Design 3**.
- Soporte para **Edge-to-Edge**.
- Estructura limpia y moderna en **Kotlin**.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Diseño:** Material 3
- **Build System:** Gradle (Kotlin DSL)

## 📱 Vista Previa
La aplicación muestra:
1. Un saludo principal: "¡Hola, Mundo! 🌍"
2. Un subtítulo: "Mi primera app Android"

## ⚙️ Requisitos
- Android Studio (Versión reciente recomendada).
- JDK 17 o superior.
- Android SDK 34 o superior.

## 🏃 Cómo ejecutar el proyecto
1. Abre Android Studio.
2. Selecciona **File > Open** y busca la carpeta del proyecto `LAB2`.
3. Espera a que termine la sincronización de Gradle.
4. Conecta un dispositivo físico o inicia un emulador.
5. Haz clic en el botón **Run** (ícono de play verde).
